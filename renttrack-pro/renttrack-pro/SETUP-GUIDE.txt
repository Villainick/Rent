# 🏠 RentTrack Pro — Setup & GitHub Deployment Guide
# (Step-by-Step, Ekdum Simple Hindi Mein)

================================================
## PART 1 — LOCAL SETUP (VS Code mein chalao)
================================================

Step 1: ZIP extract karo
  - Download ki hui ZIP file pe right-click karo
  - "Extract All" ya "Extract Here" click karo
  - Ek folder milega: "renttrack-pro"

Step 2: VS Code mein kholo
  - VS Code open karo
  - File → Open Folder → "renttrack-pro" folder select karo
  - OK click karo

Step 3: Live Server se chalao
  - VS Code mein neeche "Go Live" button dikhega (right side, status bar mein)
  - Click karo → Browser mein app khul jaayegi!
  - Pehli baar: 4-digit PIN set karo (jo bhi chahiye)

================================================
## PART 2 — GITHUB PE DEPLOY KARNA (Free Hosting)
================================================

--- Step 1: GitHub Account ---
  1. github.com pe jao
  2. "Sign up" karo (free hai)
  3. Email verify karo

--- Step 2: New Repository Banao ---
  1. GitHub pe login karo
  2. Green "New" button click karo (ya + icon → New repository)
  3. Repository name: renttrack-pro
  4. "Public" select karo
  5. "Create repository" click karo

--- Step 3: Files Upload Karo ---
  1. Repository page pe "uploading an existing file" link click karo
  2. Apne computer pe "renttrack-pro" folder kholo
  3. SAARI files select karo (Ctrl+A)
  4. GitHub pe drag & drop karo
  5. Neeche "Commit changes" click karo

--- Step 4: GitHub Pages Enable Karo ---
  1. Repository mein "Settings" tab click karo
  2. Left side mein "Pages" click karo
  3. "Branch" ke neeche "main" select karo
  4. "/(root)" select karo
  5. "Save" click karo
  6. 2-3 minute wait karo
  7. Ek link milega: https://TUMHARA-USERNAME.github.io/renttrack-pro/
  8. Yahi tumhara app ka link hai! Kisi bhi device pe kholo 🎉

--- Step 5: Mobile Pe Install Karo (PWA) ---
  Android:
  1. Chrome mein app ka link kholo
  2. 3 dots menu → "Add to Home screen"
  3. Install → Done!

  iPhone:
  1. Safari mein link kholo
  2. Share button → "Add to Home Screen"
  3. Add → Done!

================================================
## PART 3 — DATA & BACKUP
================================================

DATA KAHAN SAVE HOTA HAI?
  1. Firebase (Cloud) — Primary storage, kisi bhi device se access
  2. LocalStorage (Browser) — Offline backup, internet nahi ho tab bhi kaam kare
  3. Manual Backup — Settings → "Export JSON Backup" → Google Drive pe save karo

BACKUP KAISE KARE?
  - App mein → Settings → "Export JSON Backup" click karo
  - Ek file download hogi (renttrack_backup_DATE.json)
  - Ise Google Drive mein save karo
  - Har hafte ek baar karo — data kabhi nahi udega!

RESTORE KAISE KARE?
  - Settings → "Import Backup" → apni backup file select karo
  - Done! Saara data wapas aa jaayega

================================================
## PART 4 — APP USE KARNA
================================================

PEHLI BAAR:
  1. App khulegi → PIN set karo (4 digit)
  2. "Units" page pe jao → "+ Add Unit" click karo
  3. Har unit ki details dalo (naam, type, tenant, rent)
  4. Settings mein Electricity rate set karo (₹/unit)

MONTHLY KAAM:
  1. "Rent" page → Mahine ka month select karo → Har unit ka payment update karo
  2. "Electricity" page → Meter reading dalo → Bill auto calculate hoga
  3. "Expenses" page → Is mahine ke kharche dalo
  4. "Reports" → Monthly summary dekho

================================================
## PART 5 — PROBLEM AAYE TOH?
================================================

App nahi khul rahi?
  → index.html pe right-click → Open with Live Server

Data nahi dikh raha?
  → Settings → Import Backup → apni JSON file select karo

Firebase error?
  → Internet check karo, data LocalStorage mein safe hai

PIN bhool gaye?
  → Browser mein F12 → Console → type: localStorage.clear() → Enter
  → Page refresh karo → Naya PIN set karo
  (Note: Pehle backup zaroor le lo!)
